## 👨‍💻 Resumo Profissional

Engenheiro de Software com foco na construção de arquiteturas resilientes e soluções *offline-first*. Especializado em projetar sistemas de baixa largura de banda e otimizar fluxos de dados corporativos. Experiência comprovada no desenvolvimento de infraestrutura digital para ambientes com conectividade limitada, aplicando princípios de inovação frugal para mitigar gargalos de infraestrutura (latência humana) e democratizar o acesso tecnológico.
